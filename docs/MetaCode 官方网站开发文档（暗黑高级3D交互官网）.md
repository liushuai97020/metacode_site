# MetaCode 官方网站开发文档（暗黑高级3D交互官网）

# 1\. 项目基础信息

## 1\.1 产品名称

MetaCode — AI 智能编码助手（本地私有化 AI 代码编辑器 / Agent 开发工作台）

## 1\.2 网站定位

产品官方宣传官网，面向开发者、独立开发者、AI 编程爱好者，用于展示产品能力、介绍技术架构、展示功能截图、提供客户端下载、介绍更新日志、介绍内置 MCP/Skill 生态。

## 1\.3 技术栈（强制）

- 框架：React 19 \+ TypeScript

- 构建：Vite（极速打包）

- 样式：Tailwind CSS（暗色主题）

- 3D 引擎：Three\.js \+ @react\-three/fiber

- 动画：Framer\-motion（丝滑过渡动画）

- 图标：Lucide\-react

## 1\.4 UI 视觉规范（严格遵守）

- 主题：**纯暗黑高级风**，底色 \#070812、\#0a0b16

- 主色调：科技蓝 \#3b82f6、辅助紫 \#8b5cf6

- 渐变：蓝紫渐变、通透渐变、低饱和光晕

- 质感：磨砂玻璃、毛玻璃、模糊、柔光阴影

- 动画：流畅、轻量化、无卡顿、低帧率不掉帧

- 字体：Inter \+ 无衬线中文，简洁现代

- 风格：极简、高级、赛博科技、克制不花哨

# 2\. 网站整体结构（页面结构固定）

官网一共包含 6 个页面：

1. **首页（Landing）**：产品主打页、3D 大屏首屏

2. **功能介绍页（Features）**：全部能力展示

3. **技术架构页（Architecture）**：MCP、Skill、Agent 架构图解

4. **下载页面（Download）**：客户端下载、版本选择

5. **更新日志（Changelog）**：迭代记录

6. **关于我们（About）**：产品介绍、理念

# 3\. 页面详细开发要求（逐页写死，AI 严格执行）

## 3\.1 首页（首页必须最高质感）

### 3\.1\.1 首屏 Hero 区域

## 3\.2 功能介绍页（Features）

全部功能分六大板块，排版整齐、左右交替布局：

1. **精准编码系统**：防幻觉、锁定行号、源码映射、DOM 拾取

2. **Agent 智能体**：自动规划、自动执行、自主思考

3. **Skill 内置技能**：重构、排错、格式化、注释、优化

4. **MCP 工具市场**：联网搜索、浏览器、文件、Git、数据库

5. **本地向量记忆**：RAG、项目记忆、对话记忆

6. **暗黑极简 UI**：桌面客户端、轻量化、流畅交互

# 4\. 动画交互强制要求（必须做到）

1. **全程 60 帧流畅**，禁止卡顿、禁止重渲染抖动

2. 页面滚动：元素渐入、上浮、透明度动画

3. 卡片 hover：上浮 \+ 发光 \+ 边框渐变

4. 3D 代码球体：缓慢自转、线条流动、粒子闪烁

5. 导航栏滚动透明变暗、悬浮固化

6. 按钮点击：轻微压缩回弹

7. 所有渐变柔和、克制、不刺眼

# 5\. 网站额外功能

- 暗色永久模式（无需亮色模式）

- 顶部导航：首页、功能、架构、下载、日志、关于

- 移动端自适应、适配手机、平板、桌面

- 所有图片懒加载、优化打包体积

- 纯静态部署、无后端依赖、可直接部署静态服务器
